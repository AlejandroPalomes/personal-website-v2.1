ToDo List
A web-app task manager. It is possible to create new tasks, store them in lists, or even see the completed ones. It is also possible to track time on each task, or mark them as important. All the information is stored in localStorage, so it is safe to close the web browser.

Mario Bros
Here, I recreated the famous Super Mario Bros first level, using only JavaScript and HTML/CSS. The results are stored in localStorage, beign able to compete against a friend in the same computer. It is also possible to choose the difficulty.

Quiz Game
In collaboration with Ezequiel Garay, we developed an online multiplayer quiz game, using only front-end technologies. You can register, and the information will be stored in localStorage, with a name or photo of your choice. It is also possible to chat with other connected players, and start a common game to compete against each other.
